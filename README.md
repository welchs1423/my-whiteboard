A real-time collaborative whiteboard web application built with React, Konva.js, and Socket.io.

## 🚀 Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Graphics:** Konva.js (react-konva)
- **State Management:** Zustand
- **Backend & Real-time:** Node.js, Express, Socket.io, Multer
- **Database:** MongoDB (Mongoose)

## ✨ Features

- [x] **기본 드로잉:** 마우스 드래그를 이용한 캔버스 자유선 그리기
- [x] **색상 팔레트 & 지우개:** 선 색상 변경 및 개별 지우개 툴 구현
- [x] **선 굵기 조절:** 슬라이더를 이용한 펜과 지우개의 굵기 동적 변경 기능
- [x] **실시간 동기화 (Multiplayer):** Socket.io를 이용한 다중 접속자 간 드로잉 실시간 공유
- [x] **전체 지우기:** 캔버스 초기화 및 실시간 동기화
- [x] **이미지 다운로드:** 현재 캔버스 내용을 PNG 파일로 저장 기능
- [x] **접속자 목록:** 실시간 온라인 사용자 목록 및 인원수 표시 기능 추가
- [x] **닉네임 설정:** 접속 시 사용자 식별을 위한 닉네임 입력 기능 추가
- [x] **실시간 채팅:** 접속자 간 텍스트 메시지를 주고받을 수 있는 채팅창 구현
- [x] **도형 도구:** 사각형, 원 도구를 이용한 드래그 기반 도형 그리기 기능
- [x] **실행 취소 / 다시 실행:** Ctrl+Z / Ctrl+Y 단축키 및 버튼을 통한 Undo/Redo 기능 (실시간 동기화 포함)
- [x] **텍스트 도구:** 캔버스 클릭 후 텍스트 입력, 굵기 슬라이더로 글자 크기 조절, 실시간 동기화
- [x] **직선 / 화살표 도구:** 드래그로 직선 및 방향 있는 화살표 그리기 기능
- [x] **커스텀 색상 피커:** 프리셋 5색 외 무지개 버튼으로 어떤 색이든 자유롭게 선택
- [x] **실시간 커서 공유:** 다른 접속자의 마우스 커서 위치와 닉네임을 실시간으로 캔버스에 표시
- [x] **채팅 입력 중 표시:** 상대방이 채팅을 입력 중일 때 "OO이(가) 입력 중..." 표시
- [x] **도형 채우기 토글:** 사각형·원을 외곽선만 또는 색으로 채워서 그리기 (채우기 버튼으로 전환)
- [x] **툴 키보드 단축키:** P·E·R·C·T·L·A 키로 각 도구 즉시 전환 (각 버튼 hover 시 단축키 표시)
- [x] **반응형 캔버스:** 브라우저 창 크기 변경 시 캔버스가 자동으로 함께 늘어남
- [x] **도트 그리드 배경:** 화이트보드다운 점 격자 배경, 툴바 버튼으로 ON/OFF 토글 (PNG 저장 시 미포함)
- [x] **채팅창 접기/펼치기:** 헤더 클릭으로 채팅창 최소화, 접힌 상태에서 메시지 수 뱃지 표시
- [x] **선택 도구 (Select):** S키 또는 버튼으로 전환, 클릭으로 요소 선택·드래그로 이동, Delete/Backspace로 삭제, 선택 시 파란 점선 바운딩 박스 표시, 실시간 동기화
- [x] **레이어 관리:** 요소 선택 시 하단 패널에서 앞으로/뒤로/맨 앞으로/맨 뒤로 Z순서 조정 가능, 레이어 인덱스 실시간 표시
- [x] **JSON 내보내기/가져오기:** 캔버스 상태를 JSON 파일로 저장하고 불러오기, 협업 세션 간 공유 가능
- [x] **줌 & 패닝:** 마우스 휠로 확대/축소(커서 중심, 최대 10배), Space+드래그 또는 중간 버튼으로 캔버스 이동, Ctrl+0으로 초기화, 하단 줌 인디케이터
- [x] **복사/붙여넣기:** 선택 요소 Ctrl+C/V, 복사 시 +20px 오프셋 위치에 붙여넣기
- [x] **다중 선택:** Shift+클릭으로 추가/제거, 빈 캔버스 드래그로 박스 선택, 일괄 이동/삭제
- [x] **선 스타일:** 실선/파선/점선 전환 (툴바 세그먼트 버튼)
- [x] **투명도 조절:** 요소별 불투명도 10~100% 슬라이더, 모든 도구에 적용
- [x] **단축키 도움말:** ? 키 또는 툴바 버튼으로 전체 단축키 모달 표시
- [x] **스티커 메모:** 드래그로 크기 지정, 6가지 배경색 선택, 생성 직후 텍스트 입력 모드 진입
- [x] **이미지 삽입:** 툴바 버튼으로 파일 선택 또는 캔버스에 드래그&드롭, 원본 비율 유지
- [x] **전체 선택 (Ctrl+A):** 캔버스의 모든 요소를 한 번에 선택
- [x] **복제 (Ctrl+D):** 선택한 요소(단일/다중 모두)를 +20px 오프셋으로 즉시 복제
- [x] **정렬 도구:** 2개 이상 다중 선택 시 하단 패널에서 왼쪽/가운데/오른쪽/위/중간/아래 6방향 정렬
- [x] **잠금/해제:** 선택 요소의 이동·삭제를 잠금 (Lock 아이콘, 잠금 시 황색 표시) — 선택은 유지되어 해제 가능
- [x] **그리드 스냅:** 툴바 자석 버튼으로 28px 격자 자동 스냅 ON/OFF (그리기·이동·드래그 모두 적용)
- [x] **다크 모드:** 툴바 버튼 또는 입장 화면에서 라이트/다크 테마 전환
- [x] **텍스트/스티커 더블클릭 편집:** 선택 도구 상태에서 텍스트·스티커 메모 더블클릭 시 인라인 편집 모드 진입
- [x] **최근 사용 색상:** 최근 선택한 색상 최대 5개를 팔레트 옆에 표시, 클릭으로 즉시 재사용
- [x] **입장/퇴장 토스트 알림:** 사용자 입장·퇴장 시 화면 하단에 슬라이드 알림 표시
- [x] **삼각형 도구 (V):** 드래그로 삼각형 그리기, 채우기/선 스타일/투명도 모두 지원
- [x] **전체 보기 Zoom to Fit (F):** F키 또는 하단 버튼으로 모든 요소가 화면에 맞도록 자동 줌/위치 조정
- [x] **화살표 키 이동:** 선택된 요소를 방향키로 1px, Shift+방향키로 10px 정밀 이동
- [x] **화면 따라가기 (Follow Me):** 접속자 목록에서 👁 버튼 클릭 시 해당 유저의 뷰포트(줌·위치)를 실시간으로 따라가는 발표자 모드
- [x] **이모지 반응 (Cursor Chat):** 툴바 이모지 버튼으로 8종 이모지 선택 후 캔버스 클릭 시 해당 위치에 이모지가 1.8초간 떠오르며 모든 접속자에게 브로드캐스트
- [x] **읽기 전용 모드 (View Only):** 입장 시 "보기만 하기" 선택 시 모든 편집 도구가 비활성화되고 줌/패닝·채팅만 가능한 관람 전용 모드
- [x] **선 매끄럽게 처리 (Smoothing):** 펜 도구 선택 시 "스무딩" 버튼으로 활성화, 그린 직후 Douglas-Peucker 단순화 → Chaikin 알고리즘으로 자동 보정
- [x] **도형 자동 인식 (Smart Shape):** 펜으로 원·삼각형을 대충 그리면 자동으로 깔끔한 벡터 도형으로 변환 (닫힌 경로 감지 + 원형도 분석)
- [x] **그룹화 / 해제 (Group/Ungroup):** 다중 선택 후 Ctrl+G로 그룹화 (주황 점선 박스 표시), 그룹 클릭 시 전체 자동 선택, Ctrl+Shift+G로 해제
- [x] **프라이빗 룸 (다중 방 지원):** URL 파라미터(`?room=방이름`)를 통한 독립적인 화이트보드 공간 생성 및 공유
- [x] **데이터 영구 저장 (Persistence):** MongoDB를 연동하여 서버 재시작 및 새로고침 후에도 캔버스 데이터 유지 (디바운스 적용)
- [x] **통신 최적화 (Incremental Sync):** 전체 배열 대신 변경된 단일 요소만 브로드캐스트하여 네트워크 부하 최소화
- [x] **우클릭 컨텍스트 메뉴:** 선택된 요소 위에서 우클릭 시 복사/붙여넣기, 레이어 정렬, 그룹화, 잠금 등을 제어할 수 있는 팝업 메뉴 제공
- [x] **렌더링 최적화:** `useMemo` / `useCallback` 전면 적용 — 테마, 미니맵 계산, 선택 박스, 마우스 핸들러 등 매 렌더 재생성 방지
- [x] **파일 구조 분리 (Phase 2):** `useBoardUI` (도구·스타일·토스트·테마) 및 `useViewport` (줌·패닝·좌표 변환) 커스텀 훅 분리 — Board.tsx 로직 모듈화
- [x] **파일 구조 분리 (Phase 3):** `useCanvasEvents` 커스텀 훅 분리 — `handleMouseDown/Move/Up/DblClick` 캔버스 이벤트 핸들러 일괄 이동
- [x] **선 끝 스타일 (Line Cap):** 펜·직선·화살표 도구에 round / square / butt 세 가지 끝점 스타일 선택 (툴바 토글 버튼)
- [x] **레이어 패널:** 툴바 버튼으로 열리는 좌측 레이어 목록 — 클릭 선택, 드래그&드롭 순서 변경, 가시성 토글(눈 아이콘), 개별 잠금/삭제 지원
- [x] **프레임/슬라이드 도구:** 📐 프레임 도구로 특정 캔버스 영역을 슬라이드로 지정, 프레임 패널에서 목록 탐색 및 자동 뷰포트 이동
- [x] **PDF 내보내기:** 프레임 패널의 FileDown 버튼으로 모든 프레임을 순서대로 PDF 페이지로 저장 (jsPDF 사용)
- [x] **타임라인 플레이백:** 서버가 최대 500개의 변경 스냅샷을 기록, 타임라인 플레이어로 재생/일시정지/슬라이더 스크럽 가능
- [x] **사용자 권한 관리 (Host System):** 방에 첫 입장자가 방장(👑)으로 자동 지정, 방장은 다른 사용자를 편집/보기 전용으로 전환 가능
- [x] **WebRTC 음성 채팅:** 브라우저 네이티브 WebRTC + STUN 서버 기반 P2P 오디오 통화 — 방에서 참여/퇴장, 마이크 뮤트/언뮤트 지원
- [x] **베지어 곡선 도구:** 3클릭(시작→끝→컨트롤 포인트) 방식의 2차 베지어 곡선 그리기 지원
- [x] **속도 감응 선 굵기:** 펜 도구에서 빠르게 그릴수록 가늘고, 느리게 그릴수록 굵은 필압 효과 자동 적용
- [x] **그라디언트 채우기:** 사각형·원·삼각형·도형에 두 가지 색 선형 그라디언트 채우기 (각도·색상 설정)
- [x] **커넥터/연결선:** 두 요소를 클릭하면 곡선 화살표로 자동 연결 — 요소 이동 시 연결선 실시간 갱신
- [x] **도형 라이브러리:** 다이아몬드·육각형·별·평행사변형·십자·말풍선 6종 프리셋 도형 패널
- [x] **풀스크린 발표 모드:** 프레임을 슬라이드처럼 전체화면에서 키보드로 넘기기, 다른 사용자에게 현재 슬라이드 실시간 브로드캐스트
- [x] **레이저 포인터:** 발표 시 마우스 움직임을 빨간 점으로 다른 사용자에게 실시간 전달
- [x] **댓글 핀:** 캔버스 어디에나 클릭 한 번으로 핀 고정, 텍스트 말풍선으로 주석 달기
- [x] **리치 텍스트:** 텍스트 도구에 굵게/기울임/밑줄, 3종 폰트, 좌/중/우 정렬 서브툴바 제공
- [x] **크기 조절 텍스트박스:** 드래그로 영역 지정 후 텍스트 입력, 자동 줄바꿈 지원
- [x] **QR 코드 방 공유:** 현재 방 URL을 QR 이미지로 즉시 생성, 모달에서 확인 및 다른 기기로 스캔 참여
- [x] **SVG 내보내기:** 캔버스 요소를 SVG 벡터 파일로 저장 (rect·circle·pen·text·arrow·triangle 지원)
- [x] **캔버스 배경 이미지:** 이미지 파일을 캔버스 배경으로 투사 (35% 불투명도, 드로잉 레이어 하단)
