# 🎨 Real-time Whiteboard

React와 Konva.js를 활용한 실시간 협업 화이트보드 토이 프로젝트입니다.

## 🚀 Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Graphics:** Konva.js (react-konva)
- **State Management:** Zustand
- **Real-time:** Socket.io-client, Node.js, Express

## ✨ Features

- [x] **기본 드로잉:** 마우스 드래그를 이용한 캔버스 자유선 그리기
- [x] **색상 팔레트 & 지우개:** 선 색상 변경 및 개별 지우개 툴 구현
- [x] **선 굵기 조절:** 슬라이더를 이용한 펜과 지우개의 굵기 동적 변경 기능
- [x] **실시간 동기화 (Multiplayer):** Socket.io를 이용한 다중 접속자 간 드로잉 실시간 공유
- [x] **전체 지우기:** 캔버스 초기화 및 실시간 동기화
- [x] **이미지 다운로드:** 현재 캔버스 내용을 PNG 파일로 저장 기능
- [x] **접속자 목록:** 실시간 온라인 사용자 목록 및 인원수 표시 기능 추가
- [x] **닉네임 설정:** 접속 시 사용자 식별을 위한 닉네임 입력 기능 추가
- [x] **실시간 채팅:** 접속자 간 텍스트 메시지를 주고받을 수 있는 채팅창 구현
- [x] **도형 도구:** 사각형, 원 도구를 이용한 드래그 기반 도형 그리기 기능
- [x] **실행 취소 / 다시 실행:** Ctrl+Z / Ctrl+Y 단축키 및 버튼을 통한 Undo/Redo 기능 (실시간 동기화 포함)
- [x] **텍스트 도구:** 캔버스 클릭 후 텍스트 입력, 굵기 슬라이더로 글자 크기 조절, 실시간 동기화
- [x] **직선 / 화살표 도구:** 드래그로 직선 및 방향 있는 화살표 그리기 기능
- [x] **커스텀 색상 피커:** 프리셋 5색 외 무지개 버튼으로 어떤 색이든 자유롭게 선택
- [x] **실시간 커서 공유:** 다른 접속자의 마우스 커서 위치와 닉네임을 실시간으로 캔버스에 표시
- [x] **채팅 입력 중 표시:** 상대방이 채팅을 입력 중일 때 "OO이(가) 입력 중..." 표시
- [x] **도형 채우기 토글:** 사각형·원을 외곽선만 또는 색으로 채워서 그리기 (채우기 버튼으로 전환)
- [x] **툴 키보드 단축키:** P·E·R·C·T·L·A 키로 각 도구 즉시 전환 (각 버튼 hover 시 단축키 표시)
- [x] **반응형 캔버스:** 브라우저 창 크기 변경 시 캔버스가 자동으로 함께 늘어남
- [x] **도트 그리드 배경:** 화이트보드다운 점 격자 배경, 툴바 버튼으로 ON/OFF 토글 (PNG 저장 시 미포함)
- [x] **채팅창 접기/펼치기:** 헤더 클릭으로 채팅창 최소화, 접힌 상태에서 메시지 수 뱃지 표시
- [x] **선택 도구 (Select):** S키 또는 버튼으로 전환, 클릭으로 요소 선택·드래그로 이동, Delete/Backspace로 삭제, 선택 시 파란 점선 바운딩 박스 표시, 실시간 동기화
- [x] **레이어 관리:** 요소 선택 시 하단 패널에서 앞으로/뒤로/맨 앞으로/맨 뒤로 Z순서 조정 가능, 레이어 인덱스 실시간 표시
- [x] **JSON 내보내기/가져오기:** 캔버스 상태를 JSON 파일로 저장하고 불러오기, 협업 세션 간 공유 가능
